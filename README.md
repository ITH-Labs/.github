# ITH-Labs Organization Defaults

This repository defines the default community health files and contribution standards for all repositories under the ITH-Labs organization.

## Included Defaults
- Code of Conduct  
- Contributing Guidelines  
- Security Policy  
- Support Information  
- Pull Request Template  
- Issue Templates  
- Funding Configuration  
- Default Code Owners  

These files are automatically applied to all ITH-Labs repositories unless a specific repository provides its own overrides.

This repository does not contain project code and exists solely to centralize organization-wide standards.
